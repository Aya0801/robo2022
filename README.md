# robosys2022
ロボットシステム学練習用リポジトリ

# plusコマンド
![test](https://github.com/Aya0801/robo2022/actions/workflows/test.yml/badge.svg)

# 機能
* 標準入力から読み込んだ数字を足す

* 足した数を奇数か偶数か判定する

# ダウンロード方法
```
git clone git@github.com:Aya0801/robosys2022.git

```

# 動作例
```
seq 6 | ./plus
21.0
21.0 Odd numberes

seq 7 | ./plus
28.0
28.0 Even numberes

```
## 必要なソフトウエア
* Python
  * テスト済み: 3.7～3.10
